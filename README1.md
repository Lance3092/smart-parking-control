# smart-parking-control
parking control API for group 49 smart parking project 

Node MCU PinOuts to Components:

Component : Ultra Sonic Sensor HC-S04
D1: ECHO 
D2: TRIG
3V: VCC
GND : GND 

Component : LEDSTRIP ()
RED : D5
BLUE : D6
GREEN D7
VCC : VV 

LED with current configuration is in acive high
